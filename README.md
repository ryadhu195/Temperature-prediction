📊 Temperature Data Analysis & Visualization
📌 Overview

This project demonstrates how to analyze and visualize historical temperature data using Python. It focuses on extracting meaningful insights from raw data and presenting them through clear and informative visualizations.

🚀 What This Project Does
Loads a real-world temperature dataset
Cleans and preprocesses the data for analysis
Converts raw date values into a proper datetime format
Filters data for a specific city (e.g., Algiers)
Performs statistical analysis on temperature values
Visualizes long-term temperature trends using graphs
Saves the generated plot as an image file
📈 Key Insights Generated
🌡️ Average (mean) temperature
🔺 Maximum recorded temperature
🔻 Minimum recorded temperature
📊 Year-wise temperature trend visualization
🛠️ Tech Stack
Python – Core programming language
Pandas – Data manipulation and cleaning
NumPy – Numerical computations
Matplotlib – Data visualization
📂 Project Files
prediction.py → Main script for analysis
city_temperature.csv → Dataset used
temperature_plot.png → Generated output graph

▶️ How to Run
Install required libraries:
pip install pandas numpy matplotlib
Run the script:
python prediction.py
View results:
Console → Displays dataset info & statistics
Image → Saved plot (temperature_plot.png)

🎯 Purpose of the Project

This project is ideal for beginners who want to understand:

Data cleaning and preprocessing
Working with real-world datasets
Performing basic statistical analysis
Creating meaningful visualizations

📷 Output

A time-series graph that clearly shows temperature variations over the years, helping identify patterns and trends in climate data.
